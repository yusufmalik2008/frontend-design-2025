# Contributing to InnovateCorp Landing Page

Terima kasih sudah mau bantu improve landing page ini!

### Cara Kontribusi
1. Fork repo ini
2. Buat branch baru (`git checkout -b fitur/nama-fitur`)
3. Commit perubahan (`git commit -m 'Add: fitur baru'`)
4. Push ke branch kamu (`git push origin fitur/nama-fitur`)
5. Buka Pull Request

### Yang Boleh Dikontribusi
- Perbaikan responsivitas
- Tambah dark/light mode toggle
- Ganti gambar client logo dengan yang real
- Tambah animasi (AOS, GSAP, dll)
- Ubah jadi multi-page (About, Pricing, dll)
- Convert ke React/Vue/Svelte (kalau mau bikin versi framework)

### Yang Tidak Diizinkan
- Menghapus Tailwind CDN (kecuali diganti build tool resmi)
- Menambah framework berat tanpa diskusi dulu

### Code Style
- Pakai indentasi 2 spasi
- Tailwind class diurutkan (layout → spacing → typography → effects)
- Kalau tambah JS, taruh di folder `js/` dan import di HTML

Ayo bikin landing page ini makin cakep bareng-bareng!
