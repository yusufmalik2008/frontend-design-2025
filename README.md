# InnovateCorp – Modern Dark Landing Page

![Preview](https://i.imgur.com/3qK8vL2.png)  
*(Live demo → buka `index.html` langsung di browser)*

Sebuah landing page modern, fully responsive, dark-mode first dengan desain premium untuk perusahaan tech / SaaS / AI. Dibuat 100% dengan HTML + Tailwind CSS (via CDN) – zero build tool, langsung jalan!

### Fitur
- Dark mode elegan dengan gradient ungu-biru
- Tipografi Manrope + Material Symbols
- Hero section dengan background abstrak futuristik
- Responsive grid (mobile → desktop)
- Hover effects halus & glassmorphism cards
- CTA dan social proof section siap pakai
- Super ringan: hanya 1 file `index.html`
